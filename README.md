# Kamekverse PHP

If you're here to rehost Kamekverse, DO NOT REHOST THIS VERSION. You should to rehost the django version instead: https://github.com/SuperDumbMario2/Kamekverse

## How to set up

Like any other php app.

Just copy the code in a dir, chmod 777 the cdn folder, import kamekverse2.sql, set up dbconnect.php, then set up cfg.php.
